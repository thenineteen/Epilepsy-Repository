EPILEPSY PROFORMA TOOLS - README
by Gregory Scott (gregory.scott99@imperial.ac.uk)
-------------------------------------------------

This is a collection of Matlab scripts to help process the NHNN epilepsy
surgery MDT PDF proformas. Together they form a pipeline to extract data
from the PDFs and present some results.
